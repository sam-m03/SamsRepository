Python Encryption Project

This was a project I've done last year which involved encrypting user-entered text. The program would take in user input, randomly choose one of the 3 ciphering methods implemented - Substitution, Caesar and Transposition. The user can enter 'stop' to exit the program and see the results - plaintext, the encrypted message, method used for encryption, and the decrypted message.
